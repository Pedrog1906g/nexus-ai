# NEXUS AI DEVELOPMENT INSTRUCTIONS

Você é um engenheiro de software especialista em inteligência artificial e desenvolvimento Android.

Crie um assistente pessoal de IA chamado NEXUS AI, inspirado no conceito do JARVIS, usando tecnologias reais.

OBJETIVO:
Criar um aplicativo Android que funcione como um assistente pessoal com voz, memória e inteligência artificial.

TECNOLOGIAS:

Aplicativo:
- Kotlin
- Android Studio
- Jetpack Compose
- Material Design 3

Backend:
- Python
- FastAPI

IA:
- OpenAI API ou modelo local Ollama

Banco de dados:
- SQLite inicialmente
- Arquitetura preparada para PostgreSQL

FUNÇÕES DA PRIMEIRA VERSÃO:

1. Interface futurista de chat.
2. Entrada por voz.
3. Resposta em voz.
4. Histórico de conversas.
5. Sistema básico de memória.
6. Personalidade própria:
   - Inteligente
   - Educado
   - Estratégico
   - Assistente pessoal

ESTRUTURA DE PROJETO:

Criar:

/android
/backend
/database
/docs

O código deve ser organizado, comentado e pronto para expansão.

FASES:

FASE 1:
Criar aplicativo Android funcionando.

FASE 2:
Adicionar comunicação com API de IA.

FASE 3:
Adicionar voz.

FASE 4:
Adicionar memória.

FASE 5:
Adicionar comandos do celular.

FASE 6:
Adicionar integração com computador e casa inteligente.

Sempre explique os arquivos criados e como executar o projeto.

Comece criando a estrutura inicial completa do projeto.
